A simple console-based Tic Tac Toe game built using C++ fundamentals.
The game supports two players, allows entering player names, and uses numbers to select grid positions.

How to Play

Start the game

Run the program.

You will be asked to enter names for Player 1 and Player 2.

Game Symbols

Player 1 → X

Player 2 → O

Grid Positions

The game board uses numbers 1 to 9 to represent positions:

Making a Move

On your turn, enter a number (1–9) corresponding to the position where you want to place your symbol.

Example:

Enter 5 to place your symbol in the center.

Rules

A player cannot choose an already occupied position.

Players take turns until:

One player gets three symbols in a row (horizontal, vertical, or diagonal), or

The game ends in a draw (no empty positions left).

Winning

The game announces the winner by player name, or declares a draw.

Features

Two-player mode

Player name input

Number-based grid selection

Input validation for invalid or occupied moves

Win and draw detection

Console-based interface
